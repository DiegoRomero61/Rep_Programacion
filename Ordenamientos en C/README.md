# Comparador de Algoritmos de Ordenamiento en C

## Descripción

Aplicación desarrollada en lenguaje C que permite generar arreglos de números enteros y ordenarlos utilizando distintos algoritmos de ordenamiento. El programa compara el rendimiento de cada algoritmo mediante la medición del tiempo de ejecución y registra un historial de las operaciones realizadas.

## Características

- Creación de arreglos de tamaño definido por el usuario.
- Llenado del arreglo con números aleatorios.
- Visualización del contenido del arreglo.
- Ordenamiento mediante diferentes algoritmos.
- Medición del tiempo de ejecución de cada algoritmo.
- Registro de historial de ordenamientos realizados.
- Menú interactivo para seleccionar las operaciones.

## Algoritmos implementados

- Bubble Sort
- Bubble Sort Mejorado
- Selection Sort
- Insertion Sort
- Shell Sort
- Merge Sort
- Quick Sort

## Tecnologías

- Lenguaje C
- Code::Blocks

## Conceptos aplicados

- Arreglos
- Funciones
- Programación estructurada
- Algoritmos de ordenamiento
- Medición de tiempo de ejecución
- Manejo de archivos
- Modularización del código
- Análisis de eficiencia

## Objetivo

Este proyecto fue desarrollado con fines académicos para analizar y comparar el desempeño de distintos algoritmos de ordenamiento sobre conjuntos de datos de diferentes tamaños.
