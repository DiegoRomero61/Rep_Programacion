# Conversor de Temperaturas en C

## Descripción

Programa desarrollado en lenguaje C que convierte temperaturas entre las escalas Celsius, Fahrenheit y Kelvin mediante un menú interactivo.

## Conversiones disponibles

- Celsius → Fahrenheit
- Fahrenheit → Celsius
- Celsius → Kelvin
- Kelvin → Celsius

## Tecnologías

- Lenguaje C
- Code::Blocks
