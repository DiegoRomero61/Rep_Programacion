# Portafolio de Programación en C

Bienvenido a mi portafolio de proyectos desarrollados durante mi formación en Ingeniería en Computación.

En este repositorio encontrarás programas realizados en lenguaje C enfocados en fortalecer mis conocimientos en programación, estructuras de datos y análisis de algoritmos.

## Tecnologías

- C
- Code::Blocks
- Git
- GitHub

## Proyectos

| Proyecto | Descripción |
|----------|-------------|
| Calculadora | Operaciones matemáticas básicas mediante un menú interactivo. |
| Conversor de Temperaturas | Conversión entre Celsius, Fahrenheit y Kelvin. |
| Agenda Telefónica | Registro, búsqueda y visualización de contactos. |
| Comparador de Algoritmos | Implementación y análisis de diferentes algoritmos de ordenamiento. |

## Objetivos

- Fortalecer la lógica de programación.
- Implementar estructuras de datos.
- Analizar la eficiencia de algoritmos.
- Mejorar las buenas prácticas de programación.
