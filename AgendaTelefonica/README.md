# Agenda Telefónica en C

## Descripción

Programa desarrollado en lenguaje C que permite administrar una agenda telefónica sencilla mediante un menú interactivo.

## Funciones

- Agregar contactos
- Mostrar todos los contactos registrados
- Buscar un contacto por nombre

## Tecnologías

- Lenguaje C
- Code::Blocks

## Conceptos utilizados

- Arreglos
- Estructuras (struct)
- Ciclos
- Condicionales
- Búsqueda secuencial
- Manejo de cadenas con string.h
